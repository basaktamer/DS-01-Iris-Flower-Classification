# Iris Flower Classification

This project is a machine learning application that classifies Iris flowers into three species: **Setosa, Versicolor, and Virginica**.

## Project Overview
- **Model:** Random Forest Classifier
- **Accuracy:** 100%
- **Dataset:** UCI Iris Dataset (Kaggle)

## How to use
Adjust the sliders in the sidebar to input the sepal and petal measurements. The application will instantly calculate the most likely species and show the probability distribution across all three classes.

## Features Used
1. Sepal Length (cm)
2. Sepal Width (cm)
3. Petal Length (cm)
4. Petal Width (cm)
